How to Fix Blue Screen Error Code 0x0000007B

‚úÖ Quick Answer

Blue Screen Error 0x0000007B, also called INACCESSIBLE_BOOT_DEVICE, occurs when Windows cannot access the system drive during startup. It is usually caused by incorrect BIOS settings, corrupted drivers, or issues with the storage controller. Fixing it requires checking BIOS/UEFI settings, repairing boot files, updating storage drivers, or performing a system restore.

‚ùì Why Does 0x0000007B Occur?

The 0x0000007B BSOD is triggered when Windows loses access to the boot device. Common causes include:

  • Incorrect SATA/AHCI/IDE controller settings in BIOS
  • Corrupted or outdated storage drivers
  • Virus/malware affecting boot files
  • Failed or corrupted Windows updates
  • Hardware issues with the hard drive or SSD

üîß Fix 1: Check BIOS/UEFI Settings

  1. Restart your PC and enter BIOS/UEFI settings (usually F2, Del, or Esc during startup).
  2. Locate SATA Mode or Storage Configuration.
  3. Ensure the correct mode is selected (AHCI is recommended for modern systems).
  4. Save changes and restart your PC.

üíæ Fix 2: Update or Reinstall Storage Drivers

  1. Boot Windows in Safe Mode.
  2. Open Device Manager ‚Üí IDE ATA/ATAPI controllers or Storage Controllers.
  3. Right-click the controller ‚Üí Update driver ‚Üí Search automatically.
  4. If updating does not work, uninstall the driver and restart Windows to reinstall automatically.

üõ† Fix 3: Run Startup Repair

  1. Boot from a Windows installation media (USB/DVD).
  2. Choose Repair your computer ‚Üí Troubleshoot ‚Üí Advanced options ‚Üí Startup Repair.
  3. Follow the on-screen instructions to fix boot issues.

‚öôÔ∏è Fix 4: Check Hard Drive for Errors

  1. Boot Windows in Safe Mode or Recovery Environment.
  2. Open Command Prompt and type chkdsk C: /f /r.
  3. Press Enter and wait for the scan to complete.

üíª Fix 5: System Restore

  1. Boot into Recovery Environment ‚Üí Troubleshoot ‚Üí Advanced options ‚Üí System Restore.
  2. Select a restore point before the error started occurring.
  3. Follow prompts to restore your system files and drivers.

‚ùì Frequently Asked Questions

Can 0x0000007B be caused by viruses?

Yes, malware can corrupt boot files or storage drivers, triggering this BSOD.

Will changing SATA mode in BIOS fix the error?

Yes, selecting the correct mode (AHCI/IDE/RAID) can restore access to the boot device.

Is data lost during startup repair?

No, Startup Repair attempts to fix boot issues without affecting personal files.
