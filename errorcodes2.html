How to Fix Windows Update Error Code 0x80070422

‚úÖ Quick Answer

Windows Update Error 0x80070422 usually occurs when the Windows Update service is disabled or not running, network connections are blocked, or firewall settings prevent updates. To fix it, enable the Windows Update service, check network settings, run the update troubleshooter, and restart your PC.

‚ùì Why Does Error 0x80070422 Occur?

This error happens when Windows cannot start the update service or when system policies or firewall settings prevent updates. It may also appear if the service is set to manual or disabled, or if third-party software interferes with updates.

üîß Fix 1: Enable Windows Update Service

  1. Press Windows + R, type services.msc, and press Enter.
  2. In the Services window, locate Windows Update.
  3. Right-click ‚Üí Properties.
  4. Set Startup type to Automatic.
  5. Click Start if the service is stopped, then click Apply and OK.
  6. Restart your PC and check for updates.

üßπ Fix 2: Run Windows Update Troubleshooter

  1. Go to Settings ‚Üí Update & Security ‚Üí Troubleshoot ‚Üí Additional troubleshooters.
  2. Select Windows Update and click Run the troubleshooter.
  3. Follow the instructions to automatically detect and fix problems.

‚öôÔ∏è Fix 3: Check Network & Firewall Settings

  1. Ensure you have an active internet connection.
  2. Disable any third-party firewall temporarily.
  3. Open Control Panel ‚Üí Windows Defender Firewall ‚Üí Turn Windows Defender Firewall on or off.
  4. Turn off firewall temporarily to see if the update works.

üíæ Fix 4: Reset Windows Update Components

Sometimes, corrupted update components cause this error.

  1. Open Command Prompt as Administrator.
  2. Run the following commands one by one:
net stop wuauserv
net stop cryptSvc
net stop bits
net stop msiserver
ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
ren C:\Windows\System32\catroot2 catroot2.old
net start wuauserv
net start cryptSvc
net start bits
net start msiserver

Restart your PC and try updating again.

‚ùì Frequently Asked Questions

Can 0x80070422 occur on all versions of Windows?

Yes, it can occur on Windows 7, 8, 10, and 11 when the update service is disabled or misconfigured.

Is it safe to disable firewall temporarily?

Yes, briefly disabling the firewall for troubleshooting is safe, but remember to enable it after updates.

Will resetting Windows Update components fix the error?

Yes, it resolves most cases where update files or services are corrupted.
