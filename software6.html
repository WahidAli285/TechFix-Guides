The ‚ÄúThis app can‚Äôt run on your PC‚Äù error usually occurs because you are trying to run a 64-bit program on a 32-bit Windows, or the app is blocked by Windows security settings. To fix it, download the correct app version (32-bit or 64-bit), run it as Administrator, update Windows, and disable SmartScreen temporarily.

‚ùì Why This Error Happens?

This error is common on Windows when the system cannot start an application due to compatibility or security restrictions. It can appear for both desktop apps and games. The most common causes are:

  • Wrong version: 64-bit app on 32-bit Windows
  • Blocked by SmartScreen
  • Corrupted installation
  • Missing permissions
  • System file corruption

üîß Fix 1: Check Windows Version (32-bit vs 64-bit)

Many users download the wrong version of the app. If you have 32-bit Windows, you must install the 32-bit version.

  1. Press Windows + I to open Settings.
  2. Go to System ‚Üí About.
  3. Check System type (32-bit or 64-bit).
  4. Download the correct version of the app from the official website.

üîß Fix 2: Run App as Administrator

Some apps need admin permissions to run properly.

  1. Right-click the app icon.
  2. Select Run as administrator.
  3. If it works, right-click again ‚Üí Properties ‚Üí Compatibility ‚Üí Check Run this program as an administrator.

üîß Fix 3: Disable SmartScreen Temporarily

Windows SmartScreen can block apps that are not digitally signed or from unknown sources.

  1. Open Settings ‚Üí Privacy & security.
  2. Click Windows Security ‚Üí App & browser control.
  3. Turn off Check apps and files.
  4. Try running the app again.

Note: Enable SmartScreen again after the app works.

üîß Fix 4: Reinstall the App

If the app files are corrupted, reinstalling is the easiest fix.

  1. Open Settings ‚Üí Apps ‚Üí Apps & features.
  2. Find the app and click Uninstall.
  3. Restart your PC.
  4. Download and install the latest version from the official site.

üîß Fix 5: Run SFC and DISM (Fix System Corruption)

System corruption can block apps from running. Running SFC and DISM repairs Windows system files.

  1. Open Windows Terminal (Admin).
  2. Type:
sfc /scannow

If SFC finds issues, run DISM: 

DISM /Online /Cleanup-Image /RestoreHealth

‚ùì Frequently Asked Questions

Can a 64-bit app run on 32-bit Windows?

No. 64-bit apps only work on 64-bit Windows. You must download the 32-bit version.

Is this error caused by malware?

Sometimes yes. If your PC is infected, Windows may block apps. Run a full antivirus scan.

What if the app is safe but still not running?

Try reinstalling, running as admin, or updating Windows. If still not working, check for compatibility mode.
