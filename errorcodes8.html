How to Fix Printer Error Code 0x0000011B in Windows

‚úÖ Quick Answer

Printer Error Code 0x0000011B is a common Windows error that usually occurs after a recent update, preventing network printers from connecting properly. It is caused by security updates blocking older printer drivers, network issues, or incorrect printer settings. To fix it, you can update printer drivers, adjust Windows registry, restart printer services, or reinstall the printer.

‚ùì Why Does Printer Error 0x0000011B Happen?

Common reasons include:

  • Windows security updates blocking older printer drivers
  • Incorrect printer network configuration
  • Outdated or corrupt printer drivers
  • Printer spooler service issues
  • Firewall or antivirus blocking printer communication

üîß Fix 1: Update Printer Drivers

Most printing errors are caused by outdated or incompatible drivers.

  1. Press Windows + X ‚Üí select Device Manager.
  2. Expand Printers or Print queues.
  3. Right-click your printer ‚Üí Update driver ‚Üí Search automatically for drivers.
  4. Restart your PC and try printing again.

üõ† Fix 2: Restart Printer Spooler Service

  1. Press Windows + R, type services.msc and press Enter.
  2. Locate Print Spooler ‚Üí right-click ‚Üí Restart.
  3. Set its Startup type to Automatic for future reliability.

üíª Fix 3: Remove and Re-add the Printer

  1. Open Settings ‚Üí Devices ‚Üí Printers & scanners.
  2. Select the problematic printer ‚Üí Remove device.
  3. Click Add a printer or scanner ‚Üí follow instructions to reinstall.

  1. Press Windows + R, type regedit, press Enter.
  2. Navigate to: HKEY_LOCAL_MACHINE\System\CurrentControlSet\Control\Print
  3. Right-click ‚Üí New ‚Üí DWORD (32-bit) Value ‚Üí name it RpcAuthnLevelPrivacyEnabled.
  4. Double-click it ‚Üí set Value data to 0 ‚Üí click OK.
  5. Restart your PC and check if printing works.

üí° Fix 5: Check Firewall and Antivirus Settings

Sometimes security programs block printer communication over the network:

  • Temporarily disable firewall or antivirus.
  • Try printing. If it works, add an exception for your printer in the security software.
  • Re-enable firewall/antivirus after testing.

Yes, certain Windows security updates introduced strict authentication that can block older network printers, leading to error 0x0000011B.

Will reinstalling the printer solve the issue?

Often, yes. Removing and re-adding the printer ensures Windows installs the latest drivers and network settings.
